# 프로젝트 이름 후보 (Mini-Sentry 대체)

> 상태: **결정 보류** — 후보만 모아둔 문서. 확정 시 코드베이스 전반(`CLAUDE.md`, `knowledge/`, README, UI 타이틀 등)에서 "Mini-Sentry" 표기를 일괄 변경.
> 마지막 갱신: 2026-06-30

브라우저 JS 에러 모니터링 플랫폼(Sentry류)이라는 성격에 맞춘 이름 후보군입니다.
충돌 여부는 **빠른 웹 검색 수준**이며, 정식 상표권·도메인 등록 가용성 조사는 아닙니다.
실제 출시 전 **상표 검색(KIPRIS/USPTO) + 도메인(.com/.io) 가용성**을 별도 확인할 것.

## 유력 후보 (충돌 비교적 적음)

| 이름 | 한글 표기 | 의미 | 충돌 검토 |
|------|-----------|------|-----------|
| **ErrorTrail** | 에러트레일 | "에러의 자취를 추적" — 의미가 가장 또렷 | 🟢 에러 모니터링 제품으로 동일명 발견 안 됨 |
| **ErrorScope** | 에러스코프 | 에러를 들여다보는 관측 도구(망원경/현미경) | 🟢 동일명 모니터링 제품 발견 안 됨 (일반 조어라 소규모 사용 가능성은 있음) |

→ 현재 1순위 추천: **ErrorTrail** (의미 또렷 + 발음·한글 표기 깔끔)

## 충돌로 보류된 이름

| 이름 | 한글 표기 | 충돌 사유 |
|------|-----------|-----------|
| **BugSnap** | 버그스냅 | bugsnap.io(버그 리포팅 툴) 존재 + 대형 경쟁사 **BugSnag**과 한 글자 차이 — 가장 위험 |
| **TraceWatch** | 트레이스워치 | tracewatch.com 웹 분석 툴이 이름·도메인 선점 |
| **CatchLog** | 캐치로그 | catchlog.com(어업 e-log 관리) 선점 + JS 에러 영역에 유사한 CatchJS 존재 |

## 그 외 초기 아이디어 (미검증)

의미가 덜 직관적이라 후순위로 밀린 이름들. 충돌 검토는 아직 안 함.

- Beacon(비콘) — 블루투스 비콘과 의미가 겹쳐 애매하다는 피드백
- Pulse, Lookout, Snare(스네어), Glimpse(글림스), Tript(트립트), Watchtower(워치타워), Errly(어얼리), Sift, Tracewell
